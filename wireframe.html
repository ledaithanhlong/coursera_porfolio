Brief write-up (copy this to Coursera):
This wireframe shows the layout for my portfolio / Jurni project. The desktop view includes a top navigation, hero area with image and copy, a 4-column featured projects grid (each with image, title, description, and CTA), a skills area (pill buttons), and footer contact info. The mobile view stacks the same content into a single column, with project cards shown vertically and a simplified header. This design focuses on clear content hierarchy and responsive behavior.
